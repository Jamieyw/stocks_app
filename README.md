# Stock Market Dashboard

A real-time stock market application featuring live price tracking, intelligent search, customizable alerts, interactive charts, AI-driven market insights, daily news digests, and watchlist management.

## Features

- **Real-time Stock Prices** - Live market data and price updates
- **Intelligent Search** - Quick stock lookup and discovery
- **Interactive Charts** - Visual price history and market trends
- **AI-Powered Insights** - Smart market analysis and recommendations
- **Daily News Summary** - Automated email digests of market news
- **Personalized Alerts** - Custom notifications for price changes
- **Watchlist Management** - Track your favorite stocks
- **User Authentication** - Secure login and personalized experience

## Tech Stack

- **Next.js** - React framework for production
- **Better Auth** - Modern authentication solution
- **Inngest** - Background job processing and automation
- **Database** - Persistent data storage
